# LinkedIn Certificate Generator

A React-based web application that generates LinkedIn Learning course completion certificates in PDF format.

## ⚠️ Educational Purpose Disclaimer

This project is created for **EDUCATIONAL PURPOSES ONLY**. It is a learning exercise to demonstrate React, PDF generation, and web development concepts. The certificates generated by this application:
- Are not official LinkedIn Learning certificates
- Should not be used for professional purposes
- Are meant solely for learning and practice

## Features

- Generate custom LinkedIn Learning certificates
- Download certificates as high-quality PDFs
- Customizable certificate fields:
  - First Name
  - Last Name
  - Course Name
  - Completion Date
  - Course Length
  - Course Completion Time
  - Skills Covered

## Tech Stack

- React 18
- Vite
- TailwindCSS
- html2pdf.js
- React Router DOM

## Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build

## PDF Generation Configuration

The application uses html2pdf.js with optimized settings for high-quality certificate generation:

- 300 DPI output
- A4 landscape format
- JPEG image compression
- 4x scaling for better quality
- CORS support enabled
