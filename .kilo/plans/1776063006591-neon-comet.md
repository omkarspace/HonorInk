# Analytics, Tracking & Monitoring Plan

## Project Overview
- React 18 + Vite SPA
- React Router for routing
- Has basic ErrorBoundary component

## Plan Items

### 1. Error Tracking (Sentry)
- Install `@sentry/react` package
- Configure Sentry in `main.jsx` with DSN
- Update ErrorBoundary to report errors to Sentry
- Add error boundary metadata (user info, tags)

### 2. Page Tracking
- Add Sentry `browserTracingIntegration` for automatic page tracking
- Or implement manual page view tracking using `react-router-dom` navigation hooks
- Track page load times and performance metrics

### 3. User Event Tracking
- Create a tracking utility module (`src/lib/analytics.js`)
- Track key events: certificate generation, form submissions, button clicks
- Use Sentry's captureEvent or create custom event tracking

## Implementation Steps

1. **Install dependencies**
   ```bash
   npm install @sentry/react
   ```

2. **Configure Sentry in main.jsx**
   - Initialize Sentry with React integration
   - Add tracing integration
   - Set environment (production/development)

3. **Update ErrorBoundary**
   - Integrate Sentry's `captureException` in componentDidCatch

4. **Create analytics utility**
   - Export tracking functions for events
   - Include user context if available

5. **Add tracking to components**
   - Track certificate generation events
   - Track navigation between pages
